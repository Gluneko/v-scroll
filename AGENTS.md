# AGENTS.md

本仓库的工程纪律约定（AI 协作与人工提交都需遵守）：

- 用 `bun i` 安装依赖。
- 用最现代的 JS 写法。
- `const` 定义的常量要大写，函数用驼峰风格命名。
- 函数用 `const funcName = () => {}` 这种格式定义，不要用 `function` 定义函数。
- 合并多个连续的 `const` 声明为一个，要写 `const a = 1, b = 2, c = 3;`（而不是分多行）。
- `import` 导入函数，避免直接导入整个模块。
- 命名要极简：变量名用下划线风格，函数名用小写驼峰风格。
- 用 `await` 不要用 `.then`。
- 写纯函数，不要写类（Web Components 必须的自定义元素类除外，且类要写到最薄，逻辑全部抽成纯函数）。
- 注重代码复用，多定义函数，避免大量类似的代码结构。
- 用最新浏览器支持的原生 CSS nesting，减少代码冗余。
- `export default` 的函数/变量，除非在 `import.meta.main` 中有调用需求，否则不要先声明 `const` 变量再导出。
- 修改后运行 `./build.sh` 修复错误。
